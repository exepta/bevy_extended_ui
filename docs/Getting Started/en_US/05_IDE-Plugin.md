# IDE Plugin for Bevy Extended UI

## What the plugin does

The IDE plugin speeds up the first setup and reduces manual steps when working with `bevy_extended_ui`.

- You can create Bevy apps directly from the IDE.
- While creating `bevy_assets`, you can choose dependencies in the setup flow.
- It includes dedicated syntax support for `bevy_extended_ui`.
- You can create components from the context menu with `Beu Component`.
- Code jump is supported: use `Ctrl` + left mouse button in UI files to jump directly to the matching code location.

## Typical value

- Faster project bootstrap with less manual boilerplate.
- Fewer setup mistakes for assets and dependencies.
- Better readability and maintenance of UI files through syntax support.
- Faster navigation between HTML/templates and Rust code with `Ctrl` + left click.
